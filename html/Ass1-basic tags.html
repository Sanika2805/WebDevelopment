Assignment 1: Study of Web Technology and Basics

Objective:

To understand the core concepts of web technologies including HTML, CSS, and JavaScript, and their basic usage for building web pages.

Introduction:

Web technology includes various technologies used in the creation, management, and interaction with web applications. The fundamental languages of web development include HTML for structure, CSS for styling, and JavaScript for interactivity.

Basic HTML Example:

This section introduces the structure of an HTML document:

<!DOCTYPE html>
<html>
  <head>
    <title>Web Technology Basics</title>
  </head>
  <body>
    <h1>Welcome to Web Development!</h1>
    <p>This is a simple HTML page demonstrating basic tags.</p>
  </body>
</html>

Conclusion:

This practical gives you an introduction to web technologies. Mastering HTML, CSS, and JavaScript will allow you to create well-structured and styled web pages, with the ability to add interactivity and dynamic behavior.
